# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc].

In short: be respectful, assume good faith, give and accept constructive
feedback, focus on what is best for the project and the community.

Unacceptable behaviour includes harassment, discrimination, personal attacks,
sustained disruption, and publishing other people's private information
without consent.

## Reporting

If you experience or witness behaviour that violates this code, report it
privately to the maintainers at `sonquer@o2.pl`. Reports will be handled
confidentially. Maintainers who fail to enforce this code in good faith may
face temporary or permanent repercussions as determined by other members of
the project leadership.

## Scope

This code applies in all project spaces (issues, pull requests, discussions,
chat, in-person events) and when an individual is officially representing
the project in public spaces.
